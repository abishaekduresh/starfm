# Changelog

All notable changes to the Star FM Project (Full Stack) will be documented in this file.
See `backend/CHANGELOG.md` and `frontend/CHANGELOG.md` for detailed component changes.

## [Unreleased]

## [2.0.0] - 2026-01-03
### Added
- **Frontend Overhaul**: Replaced legacy frontend with Next.js 14 application.
- **Unified Style**: Implemented "Glassmorphism" Light Theme across all admin pages.
- **Developer Experience**: Added detailed API documentation and cURL examples to the Dashboard.

### Changed
- **API Enhancements**: Added filtering capabilities to core endpoints (`/ads`, `/channels`).
- **Project Structure**: Renamed `app` directory to `frontend` (archived old app).

## [1.0.0] - 2026-01-01
### Added
- **Initial Release**: Launched PHP Slim 4 Backend and Vanilla JS Admin Panel.
- **Core Features**: Channel Management, Ad Scheduling, Public API.
