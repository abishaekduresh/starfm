## [1.0.0] - 2026-01-03
### Added
- **Next.js Migration**:
    - Replaced vanilla HTML/JS with Next.js 14 App Router.
    - Implemented secure Authentication flow (Login/Register).
    - Created modern Dashboard with "Glassmorphism" Light Theme.
    - Implemented Channels and Ads management with sorting and filtering.
- **Features**:
    - **Dynamic Filtering**: Filter Ads by type (Banner, Video, etc.) and Channels by stream type.
    - **API Documentation**: Built-in interactive API docs on the Dashboard.
    - **Health Check**: Real-time backend connectivity check.
    - **Connectivity Alerts**: SweetAlert2 integration for user feedback.
- **UI/UX**:
    - Fully responsive mobile sidebar and navigation.
    - Optimized table layouts for mobile devices.
    - Custom 404 and Loading states.
